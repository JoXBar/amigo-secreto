# Proyecto Amigo Secreto

Este es un proyecto simple para organizar un sorteo de amigos secretos. Los usuarios pueden agregar sus nombres a una lista y luego realizar un sorteo aleatorio para elegir un amigo secreto.

## Descripción

El proyecto permite a los usuarios agregar nombres de amigos, ver una lista de amigos y realizar un sorteo para elegir aleatoriamente un amigo secreto. Es una forma divertida y fácil de organizar intercambios de regalos o simplemente un juego entre amigos o compañeros de trabajo.

## Tecnologías Utilizadas

- **HTML**: Para la estructura básica de la página.
- **CSS**: Para el estilo de la página.
- **JavaScript**: Para la funcionalidad del proyecto, incluyendo la adición de amigos y el sorteo aleatorio.
